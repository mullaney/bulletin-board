# PinBoard

A virtual corkboard to pin notes, pictures, links and more.

This was a hackathon project/proof of concept. The stack uses Node, Express, PostgreSQL, Sequelize, React and Redux. It used the [boilermaker](https://github.com/FullstackAcademy/boilermaker) template from Fullstack Academy.

## Installation/Setup

Feel free to fork this repo and improve upon it. If you want to set it up, do this:
* Fork or clone this repo
* Run `npm install` or `yarn install`
* Make sure you have PostgreSQL running and create two databases `redrag` and `redrag-test`
  - If you want to use a different name for your database. Just search the repo for 'redrag' and replace it with the name you prefer
* Run `npm seed` to seed the database
* Run `npm start-deve` to start your local server
